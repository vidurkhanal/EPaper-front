# ePaper - Digitizing Written Content for Online Assessments

![ePaper Logo](logo.png)

Welcome to the ePaper project! ePaper is an application designed to digitize written or printed text, with a special focus on quizzes and exams. This tool allows educators, trainers, and institutions to transition traditional paper-based assessments to a digital platform, making it easier for students and candidates to take exams and quizzes online.

## Features

- **Text Digitization**: Utilizes advanced Optical Character Recognition (OCR) technology with Tesseract OCR and Rust to convert handwritten or printed text into machine-readable format.

- **Online Assessment**: Create digital versions of quizzes and exams for online distribution and submission, simplifying the assessment process.

- **User-Friendly Interface**: Developed using React, Typescript, Next.js, and Tailwind CSS, ePaper offers an intuitive and modern user interface for a seamless user experience.

- **Authentication and Security**: Secure user authentication is achieved through Clerk, ensuring the privacy and confidentiality of assessment data.

## Technologies Used

- **React**: A popular JavaScript library for building user interfaces.

- **Typescript**: Provides type safety for your codebase.

- **Next.js**: A React framework for building server-rendered web applications.

- **Clerk**: A tool for user authentication and identity management.

- **Rust**: A systems programming language used for advanced OCR capabilities.

- **Tesseract OCR**: An open-source OCR engine for text recognition.

- **Tailwind CSS**: A utility-first CSS framework for styling and theming.

## Getting Started

To get started with ePaper, follow these simple steps:

1. Clone this repository to your local machine:

2. Install the required dependencies:

cd epaper
npm install

3. Start the development server:

npm run dev

4. Open your web browser and access the application at [http://localhost:3000](http://localhost:3000).

## Usage

Here's how you can use ePaper:

1. **Sign In**: Create an account or sign in if you already have one.

2. **Upload Content**: Scan or upload images or PDFs containing the handwritten or printed content you want to digitize.

3. **Start Assessment**: Create a new assessment, selecting the uploaded content as the question bank.

4. **Distribute Assessments**: Share the assessment with your students or candidates.

5. **Collect Submissions**: Gather submissions digitally, making grading and analysis easier.
